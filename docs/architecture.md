# GCP Architecture Agent Architecture

GCP architecture design agent that applies Google Cloud Architecture Framework, generates Deployment Manager/Terraform configs, designs organization hierarchy, optimizes for BigQuery and AI/ML workloads, and implements GCP security best practices.

## Domain Tools

- **design_architecture**: Design cloud architecture for GCP Architecture Agent
- **generate_template**: Generate IaC template from architecture design
- **review_architecture**: Review existing architecture against best practices
- **estimate_cost**: Estimate monthly cost for proposed architecture
- **optimize**: Recommend optimizations for existing infrastructure
